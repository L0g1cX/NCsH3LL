// ABOUT

NC Shell v2.0.77 是一个轻量级的云端渗透测试框架，提供基于 Web 的反弹 Shell 管理界面，无需本地安装任何客户端。

// 快速开始

1. 在左侧面板输入监听端口（默认 4444），点击 启动。

2. 在右侧 PAYLOADS 面板填写你的 LHOST 和 LPORT，选择适合目标系统的 Payload 类型并复制。

3. 在目标机器上执行 Payload，连接建立后终端将自动激活。

// AUTO TTY UPGRADE

勾选 AUTO TTY UPGRADE 后，框架会在检测到连接建立时自动向目标注入 TTY 升级序列，将哑 Shell 升级为全功能交互式终端，支持 Tab 补全、方向键历史、Ctrl+C 等操作。

[ ! ] 升级序列依赖目标系统已安装 python3，若无法升级请手动执行。

// TTY 升级序列

python3 -c 'import pty; pty.spawn("/bin/bash")' — 远端伪终端
export TERM=xterm-256color — 设置终端类型
stty sane — 重置行规程（确保 echo 和 \n→\r\n）
stty rows N cols M — 同步终端尺寸（自动从 xterm.js 获取）

[ ! ] Web 架构下 nc 是 subprocess（PIPE），不需要 Ctrl+Z / stty raw -echo / fg 流程。

// 注意事项

[ ! ] 本工具仅供授权渗透测试和安全研究使用。

[ ! ] 请确保在合法授权的环境中使用，遵守相关法律法规。

后端使用纯 Python + Flask + Flask-SocketIO 实现，无任何第三方中间件，所有会话状态在单节点内存中管理。
